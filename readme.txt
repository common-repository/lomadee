== Description ==

O plug-in da Lomadee para WordPress é a maneira mais prática para você divulgar as melhores ofertas do e-commerce nos conteúdos do seu blog.

A Lomadee faz a leitura de palavras-chave presentes no texto do seu post e lomadeeza, automaticamente, uma oferta relacionada ao termo, sem a necessidade de ações por parte do Publisher. Dessa forma, as ofertas são apresentadas no post já publicado (antigos e novos).

Para reconhecimento destas palavras-chave, a Lomadee utiliza categorias de produtos e serviços mais buscados no e-commerce.

Ao instalar o plug-in, você deve efetuar o login com seu usuário e senha da plataforma da Lomadee para que possamos reconhecer seu ID.

Lembramos que para uso do Plug-in Lomadee para WordPress é necessário que você tenha pelo menos um site ativo.

Se você ainda não é um Publisher Lomadee acesse [lomadee.com](http://www.lomadee.com), clique em cadastre-se e preencha seus dados.

== Installation ==

PT:

1. Faça upload do diretorio lomadee em /wp-content/plugins/
1. Ative o plugin no menu 'Plugins' no Wordpress
1. Use a opção Ferramentas->Lomadee para configurar o plugin.
1. Após o login, você poderá selecionar de quais lojas deseja exibir as ofertas.
1. O plugin irá buscar palavras chave na pagina do post e irá mostrar uma oferta relacionada.


EN:

1. Upload the directory lomadee in /wp-content/plugins/ (or install the plugin through the WordPress plugins screen directly)
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Lomadee screen to configure the plugin
1. After you have logged in, you can select which advertisers you want show offers.
1. The plugin will search keywords on your post page and show one related offer.

== Frequently Asked Questions ==

= O que é a Lomadee? =

Plataforma especializada em conectar os maiores e-commerces do mercado a produtores de conteúdo digital. Somos uma plataforma colaborativa que conecta publishers e anunciantes do mundo todo. Fundada em 2009, a empresa, que integra o Buscapé Company, traz em seu DNA a expertise que o grupo acumula em quase duas décadas de atuação no mercado digital. A palavra "Lomadee", em hindu, significa raposa. O animal, símbolo da inteligência, rapidez e perspicácia, representa as características que a Lomadee valoriza para conseguir caçar as melhores oportunidades de lucro para os publishers e anunciantes da plataforma.

= Como funciona a Lomadee? =

O nosso produto dispõe de avançada tecnologia que permite conectar quem tem espaços para divulgação com empresas que querem anunciar. Nós conectamos Publisher as maiores marcas do e-commerce.

= Como posso fazer parte da Lomadee? =

Acesse o site [lomadee.com](https://www.lomadee.com) e cadastra-se para se tornar um dos nossos Publishers. Após o cadastro você fará parte de uma rede de empreendedores digitais e poderá impulsionar, por meio do seu conteúdo, as melhores ofertas do e-commerce.
